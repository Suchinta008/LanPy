# User Guide

## Installing the App
Double-click the .exe file.

## First Time Setup
Enter your username.

## Sending Messages
Select a user from the list.
Type message and press Enter.

## Sending Files
Click the "Attach" button.
Select a file.
File will be transferred directly.

## Notes
- All users must be connected to same WiFi/LAN.
- Firewall may prompt on first launch — click Allow.
